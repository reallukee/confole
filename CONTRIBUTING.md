# Contributing

Confole è un progetto nato con finalità didattiche: imparare F#!

NOTA: **Non accetto PR**.



## Organizzazione

Confole è strutturato in tre branch fissi:

* [`main`](https://github.com/reallukee/confole/tree/main): contiene la versione corrente della codebase.
* [`next`](https://github.com/reallukee/confole/tree/next): contiene la versione successiva della codebase.
* [`experimental`](https://github.com/reallukee/confole/tree/experimental): contiene la versione **SPERIMENTALE** della codebase.



## Versionamento

Confole segue lo standard [SemVer](https://semver.org/).



## Intestazioni

Confole utilizza lo stesso modello di intestazione per ogni sorgente.

```fsharp
(*
    -------
    Confole
    -------

    Abbellisci la tua app console F# in modo funzionale

    https://github.com/reallukee/confole/

    File name   : [[NOME FILE]]

    Title       : [[TITOLO API]]
    Description : [[DESCRIZIONE API]]

    Author      : Luca Pollicino
                  (https://github.com/reallukee/)
    Version     : [[VERSIONE]]
    License     : [[LICENZA]]
*)
```
