# Contributing

Confole è un progetto nato con finalità didattiche: imparare F#!

NOTA: **Non accetto PR**.



## Organizzazione

Confole è strutturato in tre branch fissi:

* `main`: contiene la versione corrente della codebase.
* `next`: contiene la versione successiva della codebase.
* `experimental`: contiene la versione **SPERIMENTALE** della codebase.



## Versionamento

Confole segue lo standard [SemVer](https://semver.org/).
