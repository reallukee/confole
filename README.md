# Compilazione

## 0. Requisiti

### Compilazione

> [!TIP]
> .NET 8.0+ SDK consigliata!

* .NET Core 2.0+ SDK

  *Oppure*

* .NET 5.0+ SDK

## 1. Sorgente

```

```

## 2. Compila

```
cd confole/confole

dotnet build --configuration Release
```
